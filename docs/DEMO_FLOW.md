# Demo Flow

1. Open Formalio Green.
2. User logs in and lands on Home.
3. User reports an illegal dumpsite in Bafoussam.
4. A red marker appears on the Green Map.
5. User opens marker details and joins a cleanup mission.
6. User submits before/after proof.
7. Verification moves from pending to approved.
8. The marker changes to green.
9. Eco Points increase.
10. Leaderboard and impact stats update.
